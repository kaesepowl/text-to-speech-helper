The Server

### Getting started

1. Install dependencies

```bash
npm i
```

2. Set AWS credentials (Create user with Polly Full Access)

```bash
export POLLY_ACCESS_KEY_ID=yourAccessKeyIdHere
export POLLY_SECRET_ACCESS_KEY=yourSecretAccessKeyHere
```

3. Start Server

```bash
# Start Linux
npm run start
# Start windows
npm run start:win
```
